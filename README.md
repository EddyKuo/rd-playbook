# RD Playbook — 研發工程師決策手冊 2026

> 給 2026 工程師的「實作決策手冊」——當需求與設計已定,如何把程式碼寫得可讀、可測、可交付、可維運,並在 AI 輔助編碼時代守住工程品質。

本書是「決策手冊系列」的一員,與以下三本互通:

- **PM Playbook** — 產品決策
- **SA/SD Playbook** — 系統分析與設計決策
- **RD Playbook**(本書) — 研發實作決策
- **QA Playbook** — 品質保證決策
